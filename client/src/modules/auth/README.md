# Auth Module

## Purpose
Optional authentication for saving documents.

## Components
- **AuthProvider.jsx**: React context for auth state
- **LoginModal.jsx**: Login/Register modal

## Key Features
- Optional: app works fully without auth
- Auth only needed to save documents server-side
- JWT-based authentication

## Exports
- `AuthProvider` component
- `useAuth()` hook

## Dependencies
- React Context API
